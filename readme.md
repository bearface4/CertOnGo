# Cert on Go

Cert on Go is a mobile application for requesting barangay documents using an Android device.  
It is built using **Java** and **Firebase**.

---

## Features
- User authentication (login / registration)
- Request barangay certificates digitally
- Track request status
- Upload and store documents

---

## Tech Stack
- **Language:** Java
- **Platform:** Android
- **Backend:** Firebase
    - Firebase Authentication
    - Cloud Firestore / Realtime Database
    - Firebase Storage

---
